# Retriever (iOS)

GET IT ON THE APP STORE: <https://apps.apple.com/se/app/retriever-schema-app/id6754044998>

A rewrite of the NodeJS version. Now, it's completely client-side and fetches schedules directly from KronoX AJAX.

Written purely in Swift and SwiftUI to deliver a native iOS experience.

## Schedule app for Örebro University

Parses Örebro University KronoX to display a schedule with a modern UI and responsive design. Search for resources and create new schedules directly in the app. Supports dark mode.
